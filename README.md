## Punto 2
*git clone* se utiliza para copiar un repositorio remoto en una computadora local

### Sintaxis
git clone url-del-repositorio

## Punto 2a
*.gitignore* Es un archivo que se utiliza para indicar que archivos o carpetas git debe ignorar.
    Se recomienda crearlo al inicio del proyecto. Para configurarlo se agregan reglas dentro del archivo. Es conveniente usarlo para evitar subir archivos innecesarios, temporales o sensibles al repositorio

*Ignorar archivo especifico:* Para indicar cual archivo se debe ignorar solo basta escribir dentro del .gitignore cual es el archivo
-Ejemplo-
ignorado.txt
